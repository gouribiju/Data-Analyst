# 📊 IMDB Analysis

## 📌 Overview
This repository contains SQL scripts and queries designed to extract, analyze, and visualize insights from relational databases. It demonstrates proficiency in data manipulation, querying, and reporting using SQL.

##  🔍 Problem Statement
The goal of this project is to perform comprehensive data analysis on the provided datasets, understand relationships between tables, and generate meaningful insights to support decision-making.

## 🔹 Dataset Details
- **Actors** – Information about actors.
- **Directors** – Information about directors.
- **Movies** – Details of movies.
- **Movies_Directors** – Mapping between movies and directors.
- **Movies_Genres** – Mapping between movies and genres.
- **Directors_Genres** – Directors’ associated genres.
- **Roles** – Actors’ roles in movies.

## 🛠 Tools Used
- **MySQL** – Database management and querying.
- **MySQL Workbench** – SQL development and visualization.


## 📊 Dashboard Features
- Retrieve top movies, actors, and directors.
- Aggregate and filter data using `GROUP BY` and `HAVING`.
- Join multiple tables to analyze relationships between movies, actors, and directors.
- Analyze genres and director-genre associations.

## 📁File Description
- `IMDB_analysis.sql`: Contains SQL queries used to perform the analysis.

## 📜 How to Run
1. Import the dataset into MySQL.
2. Execute the queries in the `.sql` file.
3. Review insights generated from the output tables.

## 📜 Conclusion

This SQL project demonstrates the ability to manage and query relational databases efficiently. It highlights how structured queries can extract actionable insights from complex datasets, serving as a foundation for more advanced data analysis and business intelligence tasks.
