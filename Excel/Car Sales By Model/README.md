# 🚗 Car Sales Data Analysis – Excel Project

## 📊 Project Overview
This project analyzes car sales data using Microsoft Excel. It demonstrates data visualization and business insights using Pivot Tables and Charts. The goal is to explore sales performance, profit trends, and dealer contributions across different dimensions such as time, model, and dealer ID.

---

## 🎯 Objectives
The main objectives of this project were to:
- Analyze quantity sold by dealer
- Track profit trends across dates and models
- Compare yearly profit performance by dealer
- Evaluate profitability of specific car models (Hudson) across dealers

---

## 📁 Dataset Information
The dataset includes car sales-related information such as:
- Dealer ID  
- Car Model  
- Quantity Sold  
- Profit  
- Date / Year  

---

## 📌 Tasks Completed

### 1. Quantity Sold by Dealer ID
- Created a **bar chart**
- Sorted data in ascending/descending order
- Visualized total quantity sold per dealer

### 2. Profit by Date and Model
- Created a **line chart**
- Tracked profit trends over time for different car models

### 3. Profit by Year and Dealer ID
- Created a **column chart**
- Highlighted yearly profit distribution across dealers
- Formatted data series in **red**

### 4. Profit of Hudson Models by Dealer ID
- Created a **line chart**
- Removed horizontal gridlines
- Positioned legend on the right
- Customized series outline color to **green**
