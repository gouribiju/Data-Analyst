# Excel Data Cleaning Project – Fleet Equipment Inventory

## 📊 Project Overview
This project focuses on cleaning and preparing a fleet equipment inventory dataset using Microsoft Excel.  
The dataset was provided in CSV format and required multiple data cleaning steps to ensure accuracy and usability.

----------

## 🧹 Data Cleaning Tasks Performed
- Converted CSV file to XLSX format
- Adjusted column widths for better readability
- Removed empty rows using filters
- Identified and removed duplicate records
- Corrected spelling errors
- Removed extra whitespace (double spaces)
- Standardized department names using Flash Fill
- Merged split columns into a single clean column

---

## 🛠 Tools Used
- Microsoft Excel (Excel for Web)
- Data Cleaning Techniques
  - Filters
  - Remove Duplicates
  - Flash Fill
  - Find & Replace
  - Spell Check
